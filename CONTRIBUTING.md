# Contributing

Codex Skin Kit accepts contributions that can be safely redistributed.

Accepted:

- Original code.
- Self-made abstract visual assets.
- CC0 or permissively licensed assets with source and license evidence.
- Compatibility fixes, tests, and documentation improvements.

Rejected:

- Code copied from repositories without a clear license.
- Anime, film, game, or celebrity imagery.
- Commercial logos or brand assets without written permission.
- Wallpapers, fonts, screenshots, or images whose source cannot be proven.
- Changes that automatically rewrite API keys, Base URLs, proxy settings, accounts, or provider configuration.

By contributing, you confirm that you have the right to submit the content and agree to license your contribution under this repository's MIT License.
