2. Update README wording to the `thirdParty` mode.

4. Run `npm run check`.

Before switching to a verified screenshot:

1. Run macOS install/start/verify/restore with the official Codex desktop app.
2. Capture the screenshot with `verify-signal-garden-skin.sh --screenshot`.
3. Record evidence in `docs/EXPERT_REVIEW.md`.
4. Update README wording to the `verified` mode.
5. Run `npm run check`.
