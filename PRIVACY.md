# Privacy

FlowDeck does not collect or transmit analytics, conversations, credentials, cookies, API keys, selected images, or application content.

Theme processing is performed locally. FlowDeck connects only to a local application debugging endpoint on `127.0.0.1`.

External websites, including TTFlows, are opened only after an explicit user action. TTFlows websites and other external services are governed by their own privacy policies.

FlowDeck does not alter model provider settings or route model requests.
